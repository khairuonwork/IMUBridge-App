# IMUBridge-App
Real-Time IMU (Inertial Movement Unit) Streamer for Rapid AR Development

An ADB-based bridge that streams real-time IMU sensor data (Gyroscope/Accelerometer) from an Android device to a PC. A vital tool for AR and IoT developers to debug motion-tracking algorithms instantly without the friction of constant app builds.

- Zero-Build Workflow: Test spatial tracking, orientation, and gesture logic on your laptop using live hardware data without re-compiling the app.
- Ultra-Low Latency: Utilizes a wired ADB connection (via TCP/USB forwarding) to minimize lag compared to Wi-Fi-based solutions.
- Modern Android Support: Designed specifically for Android 13 (API 33) and above, handling modern runtime permissions and high-frequency sensor sampling.
- Raw & Fused Data: Access raw sensor outputs or processed Fused Location/Rotation Vector data for high-precision applications.
